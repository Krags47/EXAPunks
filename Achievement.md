## PIZZA_PARTY

**Level:** Euclid's Pizza

**Solution:**

```
NOTE XA

LINK 800
LINK 805
@REP 10
COPY 0 #POWR
COPY 1 #POWR
@END
```

**Level:** Sawayama Wonderdisc

**Solution:**

```
NOTE XA

COPY F X

LINK 800
COPY 8 #AUTH
COPY 0 #AUTH
COPY 3 #AUTH
COPY 2 #AUTH
COPY 7 #AUTH
COPY 1 #AUTH
COPY 0 #AUTH

GRAB T
REPL FILE
```

## TONER_LOW

**Level:** Zebros Copies

**Solution:**

```
LINK 800
LINK 800
@REP 4
REPL C@{1,1}
@END

@REP 4
MARK C@{1,1}
LINK 80@{0,1}
JUMP L
@END
MARK L
COPY 1 #COPY
JUMP L
```

**Note:** You have to keep running this code until you unlock the achievement. Start over if you reach the maximum cycles.
